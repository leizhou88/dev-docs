
Go to NVM website at: https://github.com/nvm-sh/

Follow installation instruction to install/Upgrade on Linux.

Usage of nvm: 

- install a specific version of node: 

// the version number is from nodejs.org
// no need to do "sudo"
# nvm install 12.15.0 

- uninstall

# nvm uninstall 12.15.0


#### nvm commands

nvm install
nvm uninstall
nvm ls
nvm use
nvm exec
